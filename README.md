# Github Battle Application

A fun application to battle your github friends. Built with React.

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve
This will serve the application using webpack-dev-server
Visit http://localhost:8080

```sh
$ npm start
```

### Build 
This bundle your js and create your index.bundle.js file in the "dist" folder to be included in your html file

```sh
$ npm run build
```

### Deploy to firebase
If you set up firebase, this will build and deploy your app for production to firebase hosting.

```sh
$ npm run deploy
```
